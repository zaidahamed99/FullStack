How to move elements in the page.

1. Relative positioning

Idea of moving an element from an original position to a different one.

2. Absolute positioning

Moves relative to the parent element

Top Right!

3. Fixed Positioning

Sticks an element to a certain part of your viewpage and lets you slide down the page as you scroll down

This is a header

4. Sticky Positioning

Allows the header to stick to its parent element
